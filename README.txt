MOVIVITAL - Guía rápida

1. Cambia las imágenes en la carpeta /img por tus propias mascotas:
   - mascota1.png → mascota triste
   - mascota2.png → mascota feliz
   - mascota3.png → mascota evolucionada

2. Para cambiar los ejercicios:
   - Edita el archivo index.html y cambia los nombres y enlaces de los botones

3. Para subir a GitHub Pages:
   a. Crea un nuevo repositorio en GitHub
   b. Sube todos estos archivos
   c. Ve a "Settings" > "Pages" > elige rama "main" y carpeta "/ (root)"
   d. Espera 1–2 minutos y tu app estará online

¡Listo! Tu app se verá como una aplicación móvil.
